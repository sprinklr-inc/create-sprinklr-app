This is a [Next.js](https://nextjs.org/) project

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000/hello-world](http://localhost:3000/hello-world) with your browser to see the default widget.

You can start editing the page by modifying `pages/hello-world.tsx`. The page auto-updates as you edit the file.
